DGI
发表于2023-05-16|图对比学习
无监督-DEEP GRAPH INFOMAX算法背景算法名称：DGI（Deep Graph Infomax），ICLR 2019 图嵌入的无监督算法大多基于random-walk构造目标函数，但它有如下缺点： 以图结构信息为代价，过分强调邻近信息 模型指标非常依赖超参数的选择 本文提出另一类基于互信息的无监督图嵌入算法，目标是最大化**局部表示(patch)与对应的图概述表示(summary)**间的互信息。 核心技术 对于 corruption function ，直接采用 A˜=A，但是 X˜是由原本的特征矩阵 X 经过随机变换得到的。也就是说，损坏的图（corrupted graph）由与原始图完全相同的节点组成，但它们位于图中的不同位置，因此将得到不同的邻近表示。 框架 最大化正样本节点和图之间的相似度，最小化负样本节点和图之间的相似。
